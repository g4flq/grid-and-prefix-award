This collectors sheet is for the difficult hard to work New Zealand BARS

This collectors sheet is shown completed. All you need is One (1) confirmed
ZL district as above to claim this collectors sheet.
If you complete all ZL1 - ZL4 you get to claim the DZL PINNACLE award on which
there is space to collect the harder New Zealand stickers E5/n, E5/s, E6 (ZK2),
ZK3, ZL0 and ZL5-9 protectorates.
These PINNACLE awards are numbered. The award and endorsements are NOT mode
specific although if you collect all in a single mode you can request that we declare
that mode on this certificate.
The achievement recognition will be posted in the OLAR table relevant mode .
